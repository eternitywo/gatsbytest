### Write a query to get all tags for your posts

```graphql
query MyQuery {
  allMdx {
    group(field: frontmatter___tags) {
      tag: fieldValue
      totalCount
    }
  }
}
```

this is guide 문서입니다.

### make a tags page template (for /tags/{tag})

```graphql
query ($tag: String) {
  allMdx(
    limit: 2000
    sort: { fields: [frontmatter___date], order: DESC }
    filter: { frontmatter: { tags: { in: [$tag] } } }
  ) {
    totalCount
    edges {
      node {
        frontmatter {
          title
        }
        slug
      }
    }
  }
}
```
